    Testing via REST Assured

  • Test via POSTMAN


  • GET: http://ca3-nharbo.rhcloud.com/CA3/api/demoadmin/getAllUsers/
    Denne bruges for at få en liste over alle users - dog er kaldet restricted til brugere med rollen Admin.
  • GET: http://ca3-nharbo.rhcloud.com/CA3/api/demouser/getcomp/OPTION/SEARCHTEXT/COUNTRY
    Denne bruges til at hente et firma ind fra cvrapi's site. Kaldet er restricted til brugere med rollen User.
  • POST: http://ca3-nharbo.rhcloud.com/CA3/api/unknown/create/USERNAME/PASSWORD
    Denne bruges til oprettelse af en bruger. USERNAME er brugernavnet og PASSWORD er det valgte password. Passwordet bliver hashet. Alle kan oprette en bruger, der er derfor ingen restrictions på dette kald.
  • POST: http://ca3-nharbo.rhcloud.com/CA3/api/demoadmin/user/USERNAME
    Denne bruges til at slette en user. USERNAME skal erstattes med en given brugers brugernavn. Metoden er resticted til brugere med rollen Admin.

    • How to use the webpages that uses the api


  • Log in as admin (username: admin - password: test) and click the All Users tab. A list of all users will be shown.
  • Log in as user (username: user - password test) and click the Company Info tab - for an example, put in "Dong" in the searchfield, select "Navn" in Søgemulighed, and at last select "DK" in Land. Click Slå Op, and a list will appear.
  • Log in as user (username: user - password test) and click the Exchange Rate tab - try and calculate an exchange rate by selecting a country in each dropdown, and type in the amount of money you want converted.
  • Open the website without logging in - click the Registration tab - type in a username and a password, and then try to log in with the selected informations.
